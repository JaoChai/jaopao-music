## Structure Tags

โครงสร้างเพลงแบบมาตรฐานที่ Suno V5 รองรับ

### Basic Structure

```markdown
[Intro] - บทนำ เพลง
[Verse] - บทเพลงหลัก
[Pre-Chorus] - ก่อนท่อนฮุค
[Chorus] - ท่อนฮุค
[Bridge] - ท่อนเชื่อม
[Outro] - บทสรุป
[Instrumental] - ท่อนดนตรี
[Break] - ท่อนพัก
```

### ตัวอย่างการใช้งาน

```markdown
[Intro]
(เสียง Rhodes ค่อยๆ ดังขึ้น)

[Verse 1]
เนื้อหาเพลงท่อนแรก

[Pre-Chorus]
ท่อนก่อนฮุค

[Chorus]
ท่อนฮุคหลัก

[Verse 2]
เนื้อหาเพลงท่อนสอง

[Chorus]
ท่อนฮุคซ้ำ

[Bridge]
ท่อนเชื่อม

[Instrumental]
(ท่อนโซโล่ saxophone)

[Chorus]
ท่อนฮุคซ้ำ

[Outro]
ค่อยๆ เบาลง
```

### Advanced Structure Tags

```markdown
[Instrumental Intro] - บทนำดนตรี
[Vocal Intro] - บทนำเสียงร้อง
[Post-Chorus] - หลังท่อนฮุค
[Interlude] - ท่อนเชื่อมกลาง
[Breakdown] - ท่อนลดจังหวะ
[Build-up] - ท่อนค่อยๆ เร้าอารมณ์
[Drop] - ท่อนปล่อยอารมณ์
[Ad-lib] - ท่อนร้องอิสระ
[Background Vocals] - เสียงร้องประกอบ
```

## Thai Language Tips

### 1. อัตราส่วนภาษาใน Lyrics

```markdown
# แนะนำ: ไทย 75% + อังกฤษ 25%

[Verse - Thai dominant]
รักที่จากลา เหมือนฝันที่จางหาย
แต่ใจยังรอคอย เธอเสมอ

[Chorus - Mix]
Baby, I still love you
รักเธอเหมือนเดิม ไม่เปลี่ยนไป
```

### 2. คำอังกฤษที่พบบ่อยในเพลงไทย

```markdown
# Love/Romance
Baby, Love, Heart, Soul, Dream, Night

# Time
Today, Tomorrow, Forever, Always, Never

# Emotions
Sad, Happy, Pain, Feel, Believe

# Places
Home, Paradise, Heaven, Earth
```

### 3. Suno V5 กับภาษาไทย

```markdown
# Suno V5 ร้องไทยได้ดี
- ออกเสียงไทยชัดเจน
- ร้องคำยาวได้
- ร้องสัมผัสได้
- ร้องเพลงไทยเดิมได้
```

### 4. ตัวอย่าง Lyrics Structure

```markdown
[Intro]
(เสียง Rhodes ค่อยๆ ดังขึ้น)

[Verse 1]
คืนนี้ฟ้าครึ้ม ฝนตกหนัก
ใจฉันก็เช่นกัน เหม็นหม่น

[Pre-Chorus]
แต่ลึกๆ ในใจ
ยังมีรักที่รอเธอ

[Chorus]
Baby, come back to me
รักเธอยังเหมือนเดิม
ไม่เปลี่ยนไป ไม่หายไป

[Verse 2]
เวลาผ่านไป แต่ใจยังรอ
ทุกคืนที่เธอไม่อยู่

[Chorus]
Baby, come back to me
รักเธอยังเหมือนเดิม
ไม่เปลี่ยนไป ไม่หายไป

[Bridge]
Forever and ever
รักเธอตลอดไป

[Outro]
(เสียง Rhodes ค่อยๆ เบาลง)
```
