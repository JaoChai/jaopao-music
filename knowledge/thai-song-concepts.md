## Thai Song Concept Patterns

### 1. รักในวัยเรียน (School Love)
**ธีมหลัก:** ความบริสุทธิ์, ความทรงจำวัยเรียน, ความรักครั้งแรก

**องค์ประกอบหลัก:**
- หน้าเรียน, ห้องเรียน, เก้าอี้คู่
- รอที่ประตูโรงเรียน, ป้ายชื่อ
- กางเกงยีนส์, เสื้อนักเรียน, กระเป๋านักเรียน
- ยามเช้า, ยามเย็น, หลังเลิกเรียน
- ความเขินอาย, ความกล้าที่ไม่กล้า

**ตัวอย่างชื่อเพลง:**
- "เพื่อนข้างห้อง"
- "รอยยิ้มหน้าโรงเรียน"
- "First Love"
- "ความลับในสมุด"
- "นั่งคู่เธอได้ไหม"

**ตัวอย่าง Hooks:**
```
"เธอคนนั้น... ที่นั่งข้างๆ ฉันเสมอ"
"รอยยิ้มของเธอ... ที่หน้าโรงเรียน"
"ฉันอยาก... นั่งคู่เธอได้ไหม"
"ความทรงจำ... ที่หน้าห้องเรียน"
```

---

### 2. รักไกล (Long Distance Love)
**ธีมหลัก:** คิดถึง, ความห่างไกล, การรอคอย, ความหวัง

**องค์ประกอบหลัก:**
- มองดวงจันทร์ดวงเดียวกัน
- นับวันนับคืน
- แผนที่, ระยะทาง
- การวิดีโอคอล, ข้อความ
- ความเหงา, ความอดทน

**ตัวอย่างชื่อเพลง:**
- "ไกลแค่ไหนก็ถึง"
- "นับวันจนได้เจอ"
- "Long Distance"
- "มองจันทร์ดวงเดิม"
- "รอวันที่ได้เจอ"

**ตัวอย่าง Hooks:**
```
"ไกลแค่ไหน... ก็คิดถึงเธอทุกวัน"
"นับวัน... นับคืน จนได้เจอเธอ"
"มองดวงจันทร์... ดวงเดียวกับเธอ"
"ไม่ว่าไกลแค่ไหน... ฉันก็รอเธอเสมอ"
```

---

### 3. คิดถึงคนเก่า (Nostalgia / Ex Lover)
**ธีมหลัก:** ทำไมถึงคิดถึง, เหลือแค่ความทรงจำ, ความเจ็บปวดที่คุ้นเคย

**องค์ประกอบหลัก:**
- ทำไมยังคิดถึง
- เหลือแค่ความทรงจำ
- ข้างถนนเดิม, ที่เดิม
- เพลงเดิม, กลิ่นเดิม
- ความเจ็บปวดที่ยังไม่จาง

**ตัวอย่างชื่อเพลง:**
- "ทำไมยังคิดถึง"
- "เหลือแค่ความทรงจำ"
- "ข้างถนนเดิม"
- "เพลงเดิม"
- "ยังลืมไม่ลง"

**ตัวอย่าง Hooks:**
```
"ทำไม... ยังคิดถึงเธอ"
"เหลือแค่... ความทรงจำ"
"ข้างถนนเดิม... ที่เราเคยเดิน"
"เพลงเดิมๆ... ที่ยังทำให้เจ็บ"
```

---

### 4. รักลับไม่ได้ (Unrequited Love)
**ธีมหลัก:** รักเธอ但她她不知道, อยู่ตรงนั้นแต่เหมือนไม่อยู่

**องค์ประกอบหลัก:**
- รักเธอ但她她不知道
- อยู่ตรงนั้นแต่เหมือนไม่อยู่
- มองแต่ไกล
- กล้าแต่ไม่กล้าบอก
- รักในใจ แต่ไม่กล้าแสดงออก

**ตัวอย่างชื่อเพลง:**
- "รักเธอ但她她不知道"
- "อยู่ตรงนั้นแต่เหมือนไม่อยู่"
- "รักคนไม่รัก"
- "รักลับ"
- "แค่เพื่อน"

**ตัวอย่าง Hooks:**
```
"รักเธอ但她她不知道... ฉันรักเธอแค่ไหน"
"อยู่ตรงนั้นแต่เหมือนไม่อยู่... ในใจเธอ"
"แค่เพื่อน... แต่อยากให้มากกว่านั้น"
"รักลับๆ... ที่เธอไม่มีทางรู้"
```

---

### 5. ขอโทษ (Apology)
**ธีมหลัก:** ผิดไปแล้ว, ขอโอกาส, ยังรักเธอ

**องค์ประกอบหลัก:**
- ผิดไปแล้ว
- ขอโอกาส
- ยังรักเธอ
- ความเสียใจ
- การเปลี่ยนแปลง

**ตัวอย่างชื่อเพลง:**
- "ขอโทษ"
- "ผิดไปแล้ว"
- "ขอโอกาส"
- "ยังรักเธอ"
- "กลับมา"

**ตัวอย่าง Hooks:**
```
"ขอโทษ... ที่ทำให้เธอเจ็บ"
"ผิดไปแล้ว... แต่ขอโอกาส"
"ยังรักเธอ... มากกว่าเดิม"
"ขอโทษ... ที่เข้าใจผิด"
```

---

### 6.ฝันถึงอนาคต (Hopeful / Future Dreams)
**ธีมหลัก:** ไปด้วยกัน, สร้างครอบครัว, วันหน้าดีกว่าวันนี้

**องค์ประกอบหลัก:**
- ไปด้วยกัน
- สร้างครอบครัว
- วันหน้าดีกว่าวันนี้
- ความหวัง
- การเริ่มต้นใหม่

**ตัวอย่างชื่อเพลง:**
- "วันหน้าดีกว่าวันนี้"
- "ไปด้วยกัน"
- "ฝันถึงเธอ"
- "เริ่มใหม่"
- "อนาคตของเรา"

**ตัวอย่าง Hooks:**
```
"วันหน้า... ดีกว่าวันนี้"
"ไปด้วยกัน... ไปยังอนาคต"
"ฝันถึงเธอ... ฝันถึงเรา"
"เริ่มใหม่... กับเธอ"
```

---

### 7. ความเหงาในเมือง (Urban Loneliness)
**ธีมหลัก:** คนเดียวในเมืองใหญ่, แสงนีออน, รถไฟใต้ดิน

**องค์ประกอบหลัก:**
- คนเดียวในเมืองใหญ่
- แสงนีออน
- รถไฟใต้ดิน
- อาคารสูง
- ความเงียบเหงา

**ตัวอย่างชื่อเพลง:**
- "คนเมือง"
- "แสงนีออน"
- "รถไฟใต้ดิน"
- "คนเดียว"
- "กลางคืนในเมือง"

**ตัวอย่าง Hooks:**
```
"คนเดียว... ในเมืองใหญ่"
"แสงนีออน... ส่องความเหงา"
"รถไฟใต้ดิน... พาไปไหน"
"กลางคืน... ที่เงียบเหงา"
```

### 8. รักตามฤดูกาล (Seasonal Love)
**ธีมหลัก:** ฝน = romantic, หนาว = nostalgic, ร้อน = energetic

#### ฤดูฝน (Rainy Season) — Romantic, Melancholic
- ฝนตก, ร่ม, ความโรแมนติก
- ความเศร้า, ความเหงา
- การรอคอย

**ตัวอย่างชื่อเพลง:**
- "ฝนตก"
- "ฝนในใจ"
- "รอฝน"
- "วันฝน"

**ตัวอย่าง Hooks:**
```
"ฝนตก... เหมือนใจฉัน"
"ฝนในใจ... ที่ไม่หยุดตก"
"รอฝน... รอคอยเธอ"
```

#### ฤดูหนาว (Winter Season) — Nostalgic, Warm
- ความอบอุ่น, ความทรงจำ
- การโอบกอด, ความใกล้ชิด
- ความเหงาที่อยากใคร่

**ตัวอย่างชื่อเพลง:**
- "หนาว"
- "โอบกอด"
- "ความอบอุ่น"
- "ฤดูหนาว"

**ตัวอย่าง Hooks:**
```
"หนาว... แต่มีเธอ"
"โอบกอด... ความอบอุ่น"
"ฤดูหนาว... ที่ไม่หนาว"
```

#### ฤดูร้อน (Summer Season) — Energetic, Happy
- ความสนุก, ความสุข
- 海滩, 阳光
- ความรักที่สดใส

**ตัวอย่างชื่อเพลง:**
- "ร้อน"
- "阳光下"
- "海滩"
- "ความสนุก"

**ตัวอย่าง Hooks:**
```
"ร้อน... เหมือนใจฉัน"
"阳光下... ที่เธออยู่"
"海滩... ที่เราเคยไป"
```

## Seasonal Themes

### ฤดูฝน (พฤษภาคม - ตุลาคม)
**Concept:** ความโรแมนติก, ความเศร้า, ความเหงา

**แนะนำ:**
- Mood: melancholic, romantic, dreamy
- Tags: `ambient`, `rainy`, `emotional`, `soft`
- Concept: รักในฤดูฝน, คิดถึง, ความเหงา

### ฤดูหนาว (พฤศจิกายน - กุมภาพันธ์)
**Concept:** ความอบอุ่น, ความทรงจำ, ความเหงาที่อยากใคร่

**แนะนำ:**
- Mood: nostalgic, warm, intimate
- Tags: `warm`, `cozy`, `intimate`, `soft`
- Concept: รักในฤดูหนาว, ความทรงจำ, ความอบอุ่น

### ฤดูร้อน (มีนาคม - เมษายน)
**Concept:** ความสนุก, ความสุข, ความรักที่สดใส

**แนะนำ:**
- Mood: energetic, happy, upbeat
- Tags: `upbeat`, `bright`, `energetic`, `summer`
- Concept: รักในฤดูร้อน, ความสนุก, ความสุข

## Quick Reference

### Mood → Concept Matching
- **Warm + Regretful** → คิดถึงคนเก่า, ขอโทษ
- **Intimate + Romantic** → รักในวัยเรียน, รักลับไม่ได้
- **Emotional + Melancholic** → ความเหงาในเมือง, รักไกล
- **Hopeful + Uplifting** → ฝันถึงอนาคต

### Tags ที่ใช้บ่อยในเพลงไทย
- `lo-fi`, `chill`, `melancholic`, `bittersweet`
- `warm`, `intimate`, `romantic`, `emotional`
- `acoustic`, `folk-pop`, `indie`, `ballad`
- `atmospheric`, `dreamy`, `soft`, `gentle`
