# Jaopao Music Factory 🎷

AI-powered music production pipeline for YouTube channels.

## Stack
- OpenClaw (Agent Framework)
- Spark/Qwen 3.5 122B (Concept + Lyrics + SEO)
- KIE API (Suno + NanoBanana 2)
- FFmpeg (Image + Audio → MP4)
- YouTube Data API (Upload)
- Neon PostgreSQL (Database)

## Architecture
1 Agent + 1 Skill + N Channels (scale by config)
